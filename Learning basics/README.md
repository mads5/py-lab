# py-lab

A structured repository for mastering Python fundamentals, problem-solving, and interview preparation.

## 📚 Course Structure

This repository is organized by learning level, with each module covering essential Python concepts through practical examples.

### **Level 1: Basics**
- `1BasicSyntax.py` - Introduction to Python syntax, variables, and basic operations
- `1DataTypes.py` - Comprehensive guide to Python data types (strings, lists, tuples, dictionaries, sets)

### **Level 2: Control & Functions**
- `2ControlFlow.py` - if/else statements, conditional logic, and decision-making
- `2Functions.py` - Function definition, parameters, return values, and scope

### **Level 3: Intermediate Concepts**
- `3LoopingTechniques.py` - For loops, while loops, nested loops, and loop optimization
- `3Logging.py` - Logging module for effective debugging and monitoring

### **Level 4: Object-Oriented Programming**
- `4PyClasses.py` - Class definition, attributes, methods, and object creation
- `4PyInheritance.py` - Inheritance, polymorphism, and method overriding
- `4PythonExceptions.py` - Exception handling, try-except blocks, and custom exceptions
- `4PythonModules.py` - Module creation, imports, and package management

### **Level 5: Advanced Techniques**
- `5Comprehensions.py` - List, dictionary, and set comprehensions
- `5FileOps.py` - File handling, reading, writing, and file operations
- `5IteratorGenerator.py` - Iterators and generators for memory-efficient code
- `5PythonVariablesCopy.py` - Deep vs shallow copy, variable references, and memory management

### **Level 6: Functional & Pattern Programming**
- `6Decorators.py` - Function decorators, syntax, and practical applications
- `6GeneratorExpressions.py` - Generator expressions and lazy evaluation
- `6RegEx.py` - Regular expressions for pattern matching and text processing

### **Level 7: Advanced OOP & Functional Patterns**
- `7ClassmethodStaticmethod.py` - @classmethod, @staticmethod decorators
- `7DecoratorExtended.py` - Advanced decorator patterns and use cases
- `7MapReduceFilter.py` - Functional programming with map, filter, and reduce
- `7Scopes.py` - Variable scoping, LEGB rule, and scope resolution

## 🎯 Learning Path

1. Start with **Level 1** to understand Python basics
2. Progress through **Level 2** for control structures and functions
3. Move to **Level 3** for intermediate looping and logging
4. Master **Level 4** with object-oriented programming concepts
5. Explore **Level 5** for advanced data handling techniques
6. Study **Level 6** for functional programming paradigms
7. Deep dive into **Level 7** for advanced patterns and optimization

## 💡 Features

- **Well-organized modules** - Each file focuses on specific Python concepts
- **Progressive difficulty** - Start from basics and advance to intermediate/advanced topics
- **Practical examples** - Real-world use cases for each concept
- **Interview preparation** - Covers common interview questions and patterns

## 🚀 Getting Started

1. Clone the repository
2. Open any `.py` file in your favorite Python IDE or editor
3. Follow the examples and experiment with the code
4. Run the files to see the output and understand the concepts
