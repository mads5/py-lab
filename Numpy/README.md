# NumPy Fundamentals

This repository contains resources and examples for understanding the basics of NumPy, a fundamental package for numerical computing in Python.

## Contents
- Introduction to NumPy
- Array Creation
- Array Manipulation
- Mathematical Operations
- Indexing and Slicing
- Reshaping Arrays
- Broadcasting

## Basic Usage
To get started with NumPy, you will need to install it. You can do this using pip:

```bash
pip install numpy
```

Once installed, you can import NumPy in your Python scripts:
```bash
import numpy as np
```

## Topics Covered
- Array Creation: Learn how to create NumPy arrays using various methods such as from lists, using built-in functions, and more.
- Array Manipulation: Understand how to manipulate arrays, including modifying, appending, and deleting elements.
- Mathematical Operations: Perform basic mathematical operations on arrays such as addition, subtraction, multiplication, and division.
- Indexing and Slicing: Access and modify subsets of your arrays using various indexing and slicing techniques.
- Reshaping Arrays: Change the shape of an array without changing its data.
- Broadcasting: Understand how NumPy handles operations between arrays of different shapes.
Feel free to explore the provided examples to deepen your understanding of these fundamental concepts!
